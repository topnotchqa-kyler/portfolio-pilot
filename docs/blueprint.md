# **App Name**: Portfolio Pilot

## Core Features:

- Homepage Creation: Implement a homepage with a brief personal introduction, skills overview, and a call to action for contact.
- Project Showcase: Develop a dedicated page showcasing previous projects with descriptions, tech stacks used, and links to live demos or GitHub repositories.
- Contact Section: Create a contact form using a service like Formspree or a similar service to handle submissions without needing a backend database.
- Blog Section: Add a blog section using markdown files transformed to html pages on build using Nextjs's static site generation capabilities
- AI-Driven Test Improvement Tool: Include a generative AI tool that recommends improvements or additions for automated tests, to provide a demonstration of the application as well as test code.
- Product Listing Page: Simulate a product listing page with dummy products.
- Product Detail Page: Simulate a product detail page for each dummy product.
- Checkout Section: Implement a mock checkout process with dummy payment information.
- Member Signup and Login: Implement a member signup and login feature that reveals a dummy member's only page.

## Style Guidelines:

- Primary color: Vivid purple (#9F50E8) to convey creativity and professionalism.
- Background color: Light gray (#EDEDED) to ensure readability and a clean, modern look.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights, providing a digital-forward feel.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and tech-savvy feel. Body font: 'Inter' (sans-serif) for readability.
- Use minimalistic line icons to represent skills, technologies, and project categories.
- Employ a grid-based layout for a structured and responsive design across all devices.
- Incorporate subtle transition effects on page load and when hovering over interactive elements to enhance user experience.
